# 🌟 知网吸星大法 - CNKI全自动下载 🔥

一个 Tampermonkey 油猴脚本，帮你在知网(CNKI)搜索结果页**全自动批量下载PDF文献**。

## ✨ 功能特点

- **全自动下载** — 自动打开文献 → 找到PDF按钮 → 点击下载 → 关闭 → 下一篇
- **断点续传** — 用 localStorage 记录已下载文献标题，刷新页面、换页都不丢，自动跳过已下载的
- **防反爬** — 随机延迟、模拟人类滚动和鼠标移动，定期自动休息
- **验证码暂停** — 检测到验证码自动暂停，手动完成后自动继续
- **自动IP登录** — 遇到登录弹窗自动点击IP登录
- **暂停/终止** — 随时可暂停或终止下载
- **可拖动面板** — 悬浮面板可拖动，不遮挡页面内容
- **下载进度统计** — 实时显示当前/累计下载数量

## 📦 安装

### 前置条件
- 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展

### 安装脚本
1. 点击 [`cnki_batch_download.user.js`](cnki_batch_download.user.js) 查看脚本
2. 点击 Raw 按钮，Tampermonkey 会自动弹出安装提示
3. 点击「安装」即可

或者手动安装：
1. 打开 Tampermonkey 管理面板 → 新建脚本
2. 复制 `cnki_batch_download.user.js` 的全部内容粘贴进去
3. Ctrl+S 保存

## 🚀 使用方法

1. 打开知网，搜索你想下载的文献
2. 在搜索结果页右下角会出现 **「🌟 知网吸星大法」** 面板
3. 点击 **「开始全自动下载」**
4. 坐等下载完成！

### 注意事项
- 首次运行如果弹窗被拦截，请点击地址栏弹窗图标 → 始终允许 → 刷新重试
- 需要有知网下载权限（通过学校VPN/图书馆等）
- 建议不要开太快，避免触发知网反爬机制

## ⚙️ 配置说明

脚本内置以下默认配置，可根据需要调整：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| delayMin | 3秒 | 每篇最少等待时间 |
| delayMax | 6秒 | 每篇最多等待时间 |
| restEvery | 20篇 | 每下载多少篇休息一次 |
| restMin | 20秒 | 休息最短时间 |
| restMax | 40秒 | 休息最长时间 |
| maxPerSession | 100篇 | 单次最多下载数量 |
